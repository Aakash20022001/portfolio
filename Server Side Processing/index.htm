Server Side Processing

Project Details

Part 1 demonstrates the use of MySQL and PHP to create on online bookmarking service in which multiple users can add to and modify their personal bookmarks.

Part 2 integrates online course materials written in XML using a custom schema definition and SQL tables with server-side parsing to HTML format.

Analysis and Design

Part 1 references two SQL tables via PHP: 'users' and 'bookmarks'. The page begins with a login form which takes a username and password. The data is checked against the 'users' table and if valid, loads the user's bookmark collection from the 'bookmarks' table. A new account is created taking a desired username, password, and email. If the username or email is already in use, an alert will be made. Otherwise, the user data will be entered into the 'user' table. If the user forgets their password, the 'forget password' form asks for their email and sends them their login details.

New bookmarks are added to the database by taking a site name, URL, and the current user. A user can add, modify, and delete entries associated with their account. Each bookmark pulls an icon from the target website on render.

Part 2 references two SQL tables via PHP: 'courses' and 'units'. The page begins with a course selection menu which name each course in the database. On loading a selected course, a title page appears which breifly describes the course and its units, with links to each one. On selecting a unit, its content is displayed with links to a multiple choice quiz and the next unit. The quizzes are parsed in PHP from an XML document to an HTML form and graded in JavaScript before displaying the correct answers and linking to the next unit.
