Web Technologies

Project Details

This is a demonstration of HTML5, CSS3, JavaScript, XML, and AJAX.

Part 1 is a resume written in XML using a custom Schema Definition and XSL Transformations.

Part 2 is an AJAX application which teaches web technologies through tutorials and quizzes.

Part 3 is a slideshow application using HTML5 canvas.

Part 4 is a utility application which perfoms various mathematical calculations.

Analysis and Design

Part 1 consists of 3 data files (XML, XSD, XSL) and a general stylesheet (CSS). The XML contains my professional summary in hierarchical format with the PARENT:child relationships below. The XSD file provides a formal definition of the above relationships, and the XSL file displays all the data in HTML format.

  • RESUME:Name,Contact,Description,Skills,Qualities,Education,Experience
    • CONTACT:Email,Phone,Mail
    • SKILLS:Skill
    • QUALITIES:Quality
    • EDUCATION:Program
      • PROGRAM:Name,Year,School,Location,Detail
    • EXPERIENCE:Job
      • JOB:Title,Year,Company,Location,Detail

Part 2 consists of a single-page web application using AJAX to navigate through 6 web development lessons and graded quizzes. It uses an external XML document containing the different content sections of the application and internal JavaScript to manipulate the page and grade the quizzes.

Part 3 uses an HTML5 canvas to create a photo slideshow. JavaScript is used to switch photos with three possible transition styles: slide, dissolve, and fade. The slideshow also has 'start', 'stop', 'back', 'next', and 'random' transport functions, and photo captions.

Part 4 is a collection of three utility tools for measurement conversion, mortgage calculation, and room mode calculation. They take user input and then output formula-based results in realtime. The measurement utility converts between metric and imperial weights, lengths, areas, and volumes. The mortgage utility calculates monthly payment amounts and interest totals based on a compound interest formula using sale price, down payment, loan term, and annual interest rate variables. The room mode utility finds the resonant frequencies of highest interest in a rectangular space using a wavelength-to-frequency formula with length, width, and height variables measured in feet.

Documentation

This assignment was tested in Internet Explorer, Opera, Firefox, and Chrome. All parts are fully functional in these browsers except for Part 2, which does not load in Internet Explorer due to the XMLHttpRequest not behaving as expected.

In Part 4 is a Room Mode calculator, which takes the dimensions of a rectangular room in feet, and outputs the axial resonant frequencies of the room below 300Hz. These frequencies have wavelengths that divisors of the room dimensions, and distort the sound inside. The equation if F=S/W, F is frequency in Hz, S is the speed of sound in feet per second, and W is the wavelength in feet. The speed of sound is 1130 feet per second in typical environments. This tool is useful in audio recording and playback contexts, as resonant frequencies will be artifically louder and quieter in certain areas of the room.
