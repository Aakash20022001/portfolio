JavaServer Faces

Project Details

This is a demonstration of JavaServer Faces for server-side processing and database management.

Part 1 tracks how many times a user visits the webpage using session tracking, and displays their IP address and timezone.

Part 2 is a picture slideshow with navagation controls and captions.

Part 3 is an online custom computer order form with hardcoded product information and session-tracked customer data.

Analysis and Design

Part 1 tracks the session using the cookieBean.

Part 2 contains a database of image names, urls, and captions.

Part 3 models popular online stores such as Amazon and Bestbuy. These sites have product pages with prices, specifications, colour/size variations, and reviews. When added to the shopping cart, the websites display the order subtotal, tax, shipping, and total amounts before checking out. My store has similar functionalities on a smaller scale.

The site requires a Component class that contains all the product data organized by type, including model, price, and specifications. The computer is represented by a Component array during customization, and then copied to another array when added to the cart, allowing further customization without accidentally affecting the cart.

Functions include getter/setters for each component, and subtotal/tax/shipping/total calculations. Ajax tags in each selection menu update the form as options are modified.
